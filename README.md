# Splitwise Service

## Description
A full-stack bill-splitting service for multiple participants.

## Tech Stack
- React
- Express
- MongoDB
- Ant Design
- Petrovich

## Features
- Create, edit, delete profiles
- Create and edit bills
- Track transactions and history
- Backend CRUD operations and business logic for calculating debts
- Structured frontend with dashboards and management pages
- Integrated Petrovich library for name validation
